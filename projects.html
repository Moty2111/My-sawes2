Индивидуальный жилой дом

Московская область 2023 142,85 м² 574 м³ строительный объем

Описание проекта

Полный цикл строительства одноэтажного дома с мансардой по индивидуальному архитектурно-строительному проекту. Проект разработан с учетом всех современных строительных норм и требований к комфортному проживанию.

Технические характеристики

  • Общая площадь: 142,85 м²
  • Строительный объем: 574 м³
  • Высота здания: 6,6 м
  • Площадь застройки: 176,11 м²
  • Этажность: 1 этаж + мансарда

Конструктивные решения

  • Фундамент: Монолитная ж/б плита с ребрами
  • Стены: Газосиликатные блоки 625x400x250 D500
  • Перекрытия: Деревянные балки 50х200 мм
  • Кровля: Металлочерепица, холодная вальцовая
  • Отделка фасада: Штукатурка

Инженерные системы

  • Водоснабжение: Центральное + газовый котел
  • Отопление: Водяное от газового котла
  • Электроснабжение: От городских сетей
  • Газоснабжение: Центральные сети
  • Вентиляция: Естественная с вытяжными шахтами

Основные материалы проекта

Материал Количество Примечание
Газосиликатные блоки 625x400x250 48,28 м³ D500, марка прочности B3.5
Металлочерепица 228,83 м² Толщина 0,5 мм, полимерное покрытие
Деревянные балки перекрытия 133 м³ Хвойные породы, 1 сорт
Арматура Ø10 А400С 1300 мм Для монолитного пояса
Ж/б перемычки 3ПБ16-37 14 шт Для оконных и дверных проемов
Цемент М500 8,5 тонн Для кладочного раствора

Особенности проекта

Монолитный ж/б пояс по всем стенам здания Утепление цоколя ЭППС 50 мм Армирование кладки согласно нормам Вентиляционные шахты из керамического кирпича Обработка деревянных элементов огнебиозащитой
